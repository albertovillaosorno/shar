# C++

This non-governing record distinguishes the standardized C++ language from
compilers, standard libraries, SDKs, engines, and SHAR-authored source.

## Review Status And Scope

- Review status: Evidence recorded.
- Evidence status: Partially verified — Repository C++ use, standards-body
  identity, and official draft sources verified; the exact language mode,
  compiler, standard library, SDK, and target remain build-specific.
- Counsel review: Not performed.
- As-of date: 2026-07-13.
- Subject class: Internationally standardized programming language.

## Covered Material

The C++ language and standards references relevant to SHAR's independently
authored Unreal runtime and native integration code.

## Repository Use And Scope

C++ is used for Unreal Engine-facing runtime and integration surfaces. A
programming language is distinct from any compiler, standard library, IDE,
engine, SDK, or published standards text. Each implementation and publication
has separate ownership and licensing.

## Provenance And Version History

The exact language dialect, compiler, standard library, Windows SDK, Unreal
Build Tool, and target platform are build-specific facts. Build and
distribution evidence must identify those versions rather than cite C++
generically.

## Authorship, Ownership, And Attribution

ISO retains rights in its published standards material subject to its terms.
Compiler and library vendors and contributors retain rights in their
implementations. SHAR contributors retain rights in independently authored
repository code subject to the repository license.

## License Or Terms Basis

No license to reproduce the complete ISO C++ standard is inferred. Public
standards-body and educational references may be cited within their terms.
Clang, Microsoft C++, standard libraries, Windows SDK components, Unreal Engine,
and other implementations require separate records and evidence.

## Distribution, Modification, And Compatibility

Source publication does not distribute the ISO standard. Binary distribution
must inventory the compiler runtime, standard library, SDK redistributables,
engine prerequisites, and notices actually delivered.

## Compliance Posture

Authored repository source confirms C++ use. Do not describe C++ itself as open
source; that characterization applies only to particular implementations or
materials.
