proprietary Radical-era libraries and tools. It is not a substitute for the
separate format records and does not grant rights in historical code, tools,
documentation, names, marks, game data, or assets.

## Covered Material

The shared historical lineage connecting Pure3D, RadCore Cement, RadSound,
RadMovie, Scrooby, TextBible, choreography, command-script, and sound-type
subjects reviewed by SHAR.

Each format or format family has a separate bibliography record. This record
covers only shared provenance, not the complete syntax, license, ownership, or
legal analysis of any individual format.

copyright notices, and compatibility facts. Public documentation must not expose
